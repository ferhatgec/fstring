# Fegeya FString
## New generation STL String implementation with powerful features.

```cpp
FString _str("Hello, ");
FString __str("World!\n");
_str.append(__str);
Print(_str);
```

## Features:
  * Flexible, hackable.
  * Included more string tools.
  * Modern.
  * Fast.
